### 基础知识

??? tip "通信、消息、信息、信号"
    - 通信：信息的传输和交换，目的是传输消息
    - 消息：通信中传输的文字、语言、图像等
    - 信息：消息中有意义的内容
    - 信号：消息的物理载体，比如点信号

---

??? tip "模拟通信与数字通信"
    - 模拟通信：信道中传输的信号是模拟信号{==（幅度连续，时间既可以是连续的，也可以是离散的）==}
    - 数字通信：信道中传输的信号是数字信号{==（幅度、时间均离散）==}

---

??? tip "有关信息的基本常识"
    - 消息中所包含的信息量与消息发生的概率密切相关，而与消息的种类和重要程度无关
    - 不同形式的消息可以包含相同的信息
    - 消息出现的概率越小，所包含的信息量就越大，消息出现的概率越大，所包含的信息量就越小，当消息出现的概率为1时，信息量为0（“太阳从东方升起”一定会出现，所以不含信息量）
    - 若干个互相独立事件构成的消息，所含信息量等于各独立事件信息量之和

---
??? tip "确知信号与随机信号"
    | 名称     | 定义  | 举例  |
    | -------- | -------------------------------------------------------------------- | ------------------------------------------ |
    | 确知信号 | 可以预先知道其变化规律的信号<br>在定义域的任意时刻都有确定的函数值     | 振幅、频率和相位都确定的正弦波 $A\cos(\omega t + \varphi)$ |
    | 随机信号 | 不可以预先知道其变化规律的信号<br>在定义域的任意时刻没有确定的函数值 | 高斯白噪声                                 |


---

??? tip "为什么一般研究的都是广义平稳随机过程"
    - 平稳随机过程：随机过程的统计特性与时间起点无关
    - 广义平稳随机过程：只限制一阶距（均值）与二阶距（方差）
    - 严平稳随机过程：任意有限维概率密度函数与时间起点无关： $P(X_{t_1}=x_1,X_{t_2}=x_2,\cdots,X_{t_k}=x_k)=P(X_{t_1+h}=x_1,X_{t_2+h}=x_2,\cdots,X_{t_k+h}=x_k)$
    - 通信系统中所遇到的信号以及噪声大都可视为宽平稳

??? tip "多径效应"
    信号通过多条路径达到接收端，每条路径由于长度（导致的时延）和衰落随时间变化，则会导致多径传播现象，多径传播对信号的影响称为多径效应


---

??? tip "衰落？快/慢衰落？"
    | 类型 | 定义/产生原因 | 特点 |
    | ---- | ------------- | ---- |
    | **衰落** | 信号在传输中因多径、遮挡、环境变化等导致{==幅度、相位、功率随机起伏==}的现象 | {==信号包络==}不稳定，时大时小 |
    | **快衰落** | 由{==多径效应==}引起，信号经过多条路径叠加，包络快速波动 | 1. 衰落速率快，周期短<br>2. 与{==多径时延==}有关<br>3. 幅度服从{==瑞利分布==} |
    | **慢衰落** | 由{==阴影效应、距离衰减、环境缓慢变化==}引起，信号均值缓慢变化 | 1. 衰落速率慢，周期长<br>2. 与{==移动速度、障碍物==}有关<br>3. 均值服从{==对数正态分布==} |

??? tip "热噪声"
    热噪声是电子器件（如电阻、半导体）内部电荷载流子（通常是电子）的随机热运动所产生的一种电噪声。它是通信系统和电子电路中最基本、最普遍存在的噪声源，也是一种白噪声。
